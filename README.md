# ViperDuel

Split-screen 2-player arcade **fighter combat** — a War Thunder–style dogfighter that runs entirely in the browser. No install, no server.

**▶ Play: https://vipershard.github.io/ViperDuel/**

## Features
- **Two modes** — Versus (head-to-head, first to 7) and Co-op (team up vs waves of AI fighters).
- **Split-screen** local 2-player on one machine.
- **5 unlockable planes** with distinct stats/models, leveled up with XP (saved in your browser).
- Forgiving arcade flight (bank-to-turn, auto-level), afterburner boost, machine guns (nose + wing), homing missiles, and floating power-ups.
- Heightmap terrain, sea, sky, clouds — crash into the ground and the pilot ejects and ragdolls across the map.

## Controls
| | Player 1 / 2 (keyboard) | Xbox controller |
|---|---|---|
| Fly | `W`/`S` pitch, `A`/`D` bank · `I`/`K`, `J`/`L` | Left stick |
| Guns | `Space` / `Enter` | `RT` or `A` |
| Missile | `Q` / `U` | `B` |
| Boost | `LShift` / `RShift` | `LT` |

Two controllers give the best experience. Built with [Three.js](https://threejs.org/) in a single static `index.html`.
